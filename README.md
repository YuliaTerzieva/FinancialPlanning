# Investment Calculator

A simple Next.js app that helps estimate how long it may take to reach a target investment portfolio.

The calculator uses inputs such as annual expenses, expected return, monthly investment, starting balance, and tax rate to estimate the portfolio amount needed and the time required to reach it.

## Why investing matters

Investing is an important part of long-term financial planning. Instead of only saving money, investing allows your money to grow over time through compound returns. Small regular contributions can become meaningful over many years.

This app is designed to make that idea easier to understand by showing how monthly investments may grow over time.

## Features

- Enter personal investment assumptions
- Calculate required portfolio size
- Estimate time needed to reach the target
- Show projected balance over time in a chart
- Explain the formulas used

## Run the project

```bash
npm install
npm run dev
```
